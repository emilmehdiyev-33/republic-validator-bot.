# 🏛 Republic AI — Validator Tracker Bot

Telegram bot for real-time monitoring of Republic AI testnet validators.

## ⚡ Features

🔎 Validator Lookup — Search by validator name, operator address or wallet address. Stake, commission, rewards, ranking and status in a single message.

🏆 Top 100 Ranking — Live leaderboard of all active validators with stake amounts and status.

🔔 Instant Alert System — Automatic Telegram notification when a validator gets jailed or starts unbonding. 24/7 monitoring.

🌐 Network Status — Latest block, block time, active/unbonding/jailed validator counts.

📊 Real-Time Data — All data is pulled directly from the Republic AI official API in real-time.

## 🚀 Commands

| Command | Description |
|---------|-------------|
| /start | Bot info |
| /ranking | Top 100 validator leaderboard |
| /network | Network status & block info |
| /alert <name> | Set jail/unbond alert |
| /alert clear | Remove all alerts |
| /help | Help |
